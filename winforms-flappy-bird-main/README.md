# Flappy Bird: Курсовая работа 🎮

Классическая аркадная игра **Flappy Bird**, разработанная на языке **C#** с использованием платформы **Windows Forms** (.NET Framework / .NET Core). Проект создан в рамках выполнения курсовой работы и демонстрирует принципы построения игровых циклов, обработки коллизий и оптимизации графического рендеринга.

## ✨ Особенности проекта

* **Pixel Art в оригинальном качестве:** Графический движок настроен на режим `NearestNeighbor` и `SmoothingMode.None`, что позволяет масштабировать пиксельные ассеты без размытия, сохраняя чёткость оригинальной графики.
* **Оптимизация рендеринга (Кэширование):** Загрузка тяжелых графических ресурсов (`Properties.Resources`) вынесена из циклического метода `OnPaint` в поля класса, что исключает задержки (фризы) во время игрового процесса.
* **Динамическая сложность:** В игру заложена система уровней. Каждые 5 очков скорость движения труб увеличивается, а зазор между ними сужается, плавно повышая сложность для игрока.
* **Сохранение рекордов:** Система автоматически отслеживает лучший результат игрока и сохраняет его локально в файл `highscore.txt`.
* **Режим разработчика (Debug Mode):** Интерактивная панель для демонстрации внутренней логики игры прямо во время защиты проекта.

## 🛠️ Управление

* `Пробел` — Прыжок (взлет птицы вверх) / Рестарт игры после экрана Game Over.
* `D` (англ.) — Включить / выключить **Режим разработчика (Debug Mode)**.

## ⚙️ Режим разработчика (Debug Mode)

При нажатии клавиши **`D`** на экран выводится скрытый интерфейс отладки:
1.  **Визуализация хитбоксов:** Птица и трубы обводятся красными контурами, наглядно демонстрируя работу алгоритма проверки столкновений (`CheckCollisions`).
2.  **Телеметрия (Live Stats):** В правом верхнем углу появляется полупрозрачная оверлей-панель с выводом технических параметров в реальном времени:
    * Текущая координата птицы по оси Y (`Bird Y`).
    * Текущая скорость падения/взлета (`Velocity`).
    * Размер зазора между трубами в пикселях (`Gap Size`).
    * Текущая скорость движения игрового мира (`Pipe Spd`).

## 📐 Технические детали реализации

* **Размер окна:** Фиксированный ($400 \times 600$ пикселей, `DoubleBuffered = true` для устранения мерцания экрана).
* **Размер труб:** Используются оригинальные ассеты размером $60 \times 400$ пикселей. Отрисовка реализована методом смещения, что исключает деформацию текстур и появление невидимых зон.
* **Хитбоксы:** Для повышения отзывчивости управления реальный хитбокс птицы (`BirdHitboxSize = 40`) сделан чуть меньше визуального размера спрайта (`BirdVisualSize = 55`), что прощает игроку незначительные касания краев препятствий.
