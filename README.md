# Flappy Bird (WinForms Edition)

Классическая аркадная игра Flappy Bird, реализованная на платформе Windows Forms (C#). Мой самостоятельный игровой проект, разработанный в рамках курсовой работы.

## 🎮 Реализованный функционал и механики
* **Физика и управление:** Настроена гравитация (постоянное падение птицы вниз) и импульс прыжка вверх при нажатии на клавишу `Space` (Пробел).
* **Бесконечная генерация:** Написан алгоритм непрерывного создания препятствий (класс `Pipe`) с динамически изменяющейся высотой труб. Отработан алгоритм удаления старых труб за экраном для оптимизации памяти.
* **Адаптивная сложность:** Каждые 5 набранных очков увеличивается уровень игры. С каждым уровнем скорость движения труб (`pipeSpeed`) растет, а свободный зазор для пролета (`currentGap`) уменьшается.
* **Сохранение прогресса:** Реализована система локального хранения лучшего результата. Рекорд игрока записывается и считывается из файла `highscore.txt`.
* **Кастомная 2D-графика (GDI+):** Настроен плавный рендеринг без мерцания экрана благодаря включению свойства `DoubleBuffered`. Добавлена динамическая смена окружения (цвет неба и труб меняется в зависимости от текущего уровня). Разработан оверлей для состояния Game Over.

## 🛠 Технологии и стек
* **Язык программирования:** C# 
* **Платформа:** .NET / Windows Forms
* **Графический движок:** GDI+ (компонент `System.Drawing.Graphics`)
